## ðŸ“– About FinRobot-AF

FinRobot-AF is a sophisticated multi-agent framework designed specifically for financial analysis and market forecasting. Built on Microsoft's Agent Framework, it provides:

- **Pre-configured Financial Agents**: Market analysts, investors, statisticians
- **Flexible Workflows**: From simple single-agent to complex multi-agent orchestration
- **Rich Data Sources**: FinnHub, Yahoo Finance, SEC filings, and more
- **Production Ready**: Built-in observability, error handling, and state management
